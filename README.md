# Meu primeiro projeto Flask

Este é o meu primeiro projeto usando Python e Flask.

Aprendi a criar uma aplicação web simples que recebe um nome pela URL
e mostra esse nome na mensagem da página.

## O que o projeto faz

- Mostra uma mensagem na tela
- Recebe um nome pela URL
- Exibe esse nome dentro da mensagem

Exemplo de uso:

http://IP_DO_SERVIDOR:5000/Oi

## Tecnologias utilizadas

- Python 3
- Flask
- Git
- GitHub
- AWS EC2 (para testes)

## Como executar o projeto

1. Instale o Flask:
pip install flask

2. Execute o arquivo:
python app.py

3. Acesse no navegador:
http://localhost:5000
